# HTMLy ben
Tactile theme by Jason Long, Dirmanhana ported to HTMLy.

## Installations 
 -  Upload and extract the zip file into themes directory.
 -  Rename the extracted folder to `ben`.
 -  Activate it from HTMLy panel.
